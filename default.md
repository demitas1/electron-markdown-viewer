# Welcome to Markdown Viewer

GitHub互換のマークダウンビューアへようこそ。

## 機能

このアプリケーションは以下の機能を提供します：

- **GitHub Flavored Markdown (GFM)** に対応
- **シンタックスハイライト** - コードブロックの美しい表示
- **テーマ切り替え** - ライト/ダーク/システム設定対応
- **ファイルを開く** - メニューから別のマークダウンファイルを開く

## 使い方

### ファイルを開く

**File** → **Open Markdown File...** からマークダウンファイルを選択できます。

ショートカットキー: `Ctrl+O` (または macOS では `Cmd+O`)

### テーマ切り替え

**View** メニューから以下のテーマを選択できます：

- **Light Theme** - ライトテーマ
- **Dark Theme** - ダークテーマ
- **Follow System Theme** - システム設定に従う（デフォルト）

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$
